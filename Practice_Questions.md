# CSIT-605 Python Programming – Viva Questions

---

## Practice Questions for Students

1. Write a program to check if a given number is even or odd.
2. Write a program to check if the given number is positive, negative or zero.
3. Write a program to find the largest of 3 numbers.
4. Write a program to check whether a person is eligible to vote (age => 18).
5. Write a program to find the factorial of a given number.
6. Write a program to check if a given number is prime or not.
7. Write a program to print the Fibonacci series up to n terms.
8. Write a program to find the sum of all numbers in a list.
9. Write a program to reverse a string.
10. Write a program to check if a string is a palindrome or not.
11. Write a program to find the largest element in a list.
12. Write a program to count the number of vowels in a string.
13. Write a program to check if a given year is a leap year or not.
14. Write a program to check if the character is a vowel or consonant.
15. Write a program to assign grade depending on the marks obtained by a student (A for 90-100, B for 89-75, C for 74-60, D for 59-30, F for below 30).
