# Experiment 01 — Python Installation, IDE Setup & Hello World

**Subject:** CSIT-605 Python Lab  
**Location:** RGPV, Bhopal

---

## 1. Aim

To install Python, configure an IDE, understand the Python interpreter, and write the first `Hello, World!` program.

## 2. Theory

| Concept | Description |
|---------|-------------|
| **Python** | A high-level, interpreted, general-purpose programming language known for its simple syntax |
| **Interpreter** | Executes Python code line by line at runtime (unlike compiled languages) |
| **IDE** | Integrated Development Environment — provides editor, debugger, and runner in one tool |
| **`print()`** | Built-in function that outputs text to the console |

- Python source files use the `.py` extension.
- The Python interpreter can be run interactively (REPL) or execute a `.py` file.
- Popular IDEs: VS Code, PyCharm, IDLE (bundled with Python).
- `Hello, World!` is the universal first program used to verify a working environment.

## 3. Implementation Code

```python
print("Hello, World!")
```

## 4. Expected Output

```text
Hello, World!
```

## 5. Viva / Discussion Questions

1. **Interpreted vs Compiled:** What is the difference between an interpreted and a compiled language? Which category does Python fall into?
2. **REPL:** What does REPL stand for? How do you start the Python interactive shell?
3. **`print()` function:** What happens if you call `print()` with no arguments?
4. **File Extension:** What extension do Python source files use, and why?
5. **Version Check:** What command do you run in a terminal to verify the installed Python version?
